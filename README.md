# BraveEve

BraveEve is a conversational distress screening chatbot developed for female cancer patients based on the NCCN Distress Thermometer (v1.2025).

## Technologies

- Python
- Streamlit
- Scikit-learn
- Pandas
- NLP

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
streamlit run chatbot.py
```
